# pubg-api-wrapper

> A simple wrapper for the PUBG API.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i pubg-api-wrapper --save-dev
```

## Usage

```js
var pubg-api-wrapper = require('pubg-api-wrapper');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
