# pubg-api-promise

> A simple, promise-based wrapper for the PUBG API.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i pubg-api-promise --save-dev
```

## Usage

```js
var pubg-api-promise = require('pubg-api-promise');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
