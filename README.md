# pubg-api-promise

> A simple, promise-based wrapper for the PUBG API.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i pubg-api-promise
```

## Usage

```js
var pubg = require('pubg-api-promise');
```
